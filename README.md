# DSP Toolkit

This project demonstrates:
- Signal generation
- FFT analysis
- Filtering

1. Signal generation and noise addition: 
        - a sine signal is generated
        - a noisy signal added to sine signal
        - signal_genarator.py

2. FFT analysis:
        - frequency of the signal is analysed
        - FFT is carried out
        - transform.py

3. Digital filtering:
        - filtering of the signal is carried out
        - different types of filters analysed -- low pass, high pass, bandpass, moving average
        - Butterworth filter is used
        - filters.py

***all plots generated using outputs.py & saved in /results folder
